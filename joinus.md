## 核心架构部招聘

### 系统开发工程师（偏 CDN 方向）

#### 岗位职责：

* 负责 UPYUN CDN 系统设计和研发工作
* 负责 NGINX / TrafficServer 模块开发工作

#### 岗位要求：

* 有较大型的 C/C++ 项目开发经验
* 熟悉 Linux 平台编程和网络编程
* 扎实的系统层面编程能力
* 熟悉 HTTP 协议
* 熟悉 Lua 语言

> 加分项：对 NGINX 或 TrafficServer 有一定开发经验

### 系统开发工程师（偏 API 方向）

#### 岗位职责：

* 负责 UPYUN API 系统研发工作
* 负责 UPYUN SDK 开发和维护工作

#### 岗位要求：

* 熟悉 Linux 平台编程和网络编程
* 扎实的系统层面编程能力
* 熟悉 HTTP 协议和 REST 接口设计
* 熟悉 OpenResty 框架
* 熟悉 Python, Lua, Golang 等多门编程语言

### 系统开发工程师（偏 DNS 方向）

#### 岗位职责：

* 负责 UPYUN DNS 系统底层平台研发工作

#### 岗位要求：

* 熟悉网络编程，熟悉 Linux 操作系统
* 有丰富的高并发系统设计和开发经验
* 有丰富的类 BIND 系统使用和开发经验，精通 DNS 协议
* 能熟练使用 C 语言进行日常开发工作
* 熟悉 Python, Lua, Golang 等其中一种编程语言

### 安全开发工程师

#### 岗位职责：

* 负责 UPYUN WAF 系统设计和研发工作

#### 岗位要求：

* 熟悉 NGINX 和 Lua 语言
* 熟悉常用的网络测试平台和渗透测试工具
* 熟悉常见的 Web 安全漏洞，有一定 WAF 系统的开发经验
* 对安全领域有一定的钻研精神

### 云存储开发工程师

#### 岗位职责：

* 负责 UPYUN 分布式云存储系统研发工作

#### 技能要求：

* 熟练使用 Erlang 语言，熟悉函数式编程
* 熟悉网络编程，对于 HTTP 协议比较了解，会使用 tcpdump 等工具分析网络包
* 至少熟悉一种 C 系列语言 (C/C++/Golang)，并且了解 Ruby/Python/Lua 等其中一门脚本语言
* 熟悉常用的分布式模型，带中心节点和不带中心节点的
* 熟悉常用的设计模式，有一定的系统编程经验
* 有一定的钻研精神，能够根据日志信息分析出问题点
* 了解或者熟悉 Riak, Cowboy, Redis 等组件

### 云处理开发工程师

#### 岗位职责：

* 负责云处理平台研发工作

#### 岗位要求：

* 扎实的 C 语言编程基础及算法优化能力
* 至少能够熟练使用一门脚本语言（Python, Ruby, Lua 等）进行日常开发
* 对 GraphicsMagick、ImageMagick、FFmpeg 等开源项目有一定开发和调优经验
* 熟悉服务端编程，熟悉 Linux 多线程、多进程模型

### 流媒体开发工程师

#### 岗位职责：

* 负责 UPYUN CDN 流媒体解决方案研发工作

#### 岗位要求：

* 有丰富的高并发系统设计和开发经验，熟悉 CDN 行业
* 扎实的 C/C++ 语言编程基础及算法优化能力，熟悉 Linux 平台编程和网络编程
* 精通 RTMP 和 HLS 协议，有丰富的流媒体协议开发经验
* 熟悉 Python, Lua, Golang 等其中一门编程语言
